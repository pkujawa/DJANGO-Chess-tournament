Potrzebne jest zaistalowanie django oraz django-rest framework.
Należy dodać kilka obiektów do bazy danych, w tym superusera.
Aplikację uruchamia sie poprzez:
    python manage.py runserver
Aplikacja daje możliwość wysyłania zapytań do REST API i oglądania rezultatów w django-restframeworkowej wersji.
Cześć linków jest wygenerowana dla obiektów z id=1, należy więc podmienić id na inne, odpowiednie dla istniejącego
obiektu.
